# Weather-App
A responsive weather application built with React that fetches real time weather data using a weather API

## Features

- Search weather by city
- Real-time weather data
- Temperature
- Weather condition
- Humidity
- Wind speed
- Weather icon
- Loading state
- Responsive design

## Tech Stack

- React
- JavaScript (ES6+)
- CSS3
- HTML5
- Vite
- Weather API
